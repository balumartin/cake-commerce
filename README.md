# CakeCommerce

## Full-stack e-commerce application built with a modern React + Node.js stack.



### 🧩 Project Structure
cake-commerce/
- backend/   # Node.js + Express + Prisma API
- frontend/  # React + Vite (planned)

---

### 🧱 Tech Stack

### Backend
- Node.js
- Express
- PostgreSQL (Neon)
- Prisma ORM
### Frontend (planned)
- React
- Vite
- Tailwind
- TanStack Query

---

### 🚀 Features (planned)

- JWT Authentication (access + refresh token)
- Role-based authorization
- Product CRUD
- Order management
- API filtering / pagination

---

### ⚙️ Architecture

- Layered backend architecture:
  - Controller → Service → Repository
- Separation of concerns
- REST API design
- Prisma as data access layer
